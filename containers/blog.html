Bootstrap Grid System Cheat Sheet

Cheat Sheet

  • Grid System: Use .row and .col-* inside a .container or .container-fluid.
  • Columns per Row: A single .row can contain up to 12 columns. If more columns are added, they wrap to the next row.
  • Nesting: Rows can be nested inside columns to create complex layouts. Nested rows also have a 12-column system.
  • Breakpoint Classes: Use classes like .col-sm-*, .col-md-*, etc., for responsive design.
