Fixed-width Container

This is a fixed-width container. Its width changes based on the screen size but does not exceed the maximum defined width.

For example: On a small screen, this container will have a width of 540px, and on a large screen, it can grow to 1320px.

Fluid Container

This is a fluid container. It stretches to cover the entire width of the browser window, no matter the size of the viewport.

For example: Even if you resize the browser, this container will always occupy the full width.

Fixed-width Example

This container stays centered with a maximum width. Use it when you want your content to look neat and constrained.

Fluid Example

This container stretches across the screen. It's ideal for layouts like banners or full-width sections.
